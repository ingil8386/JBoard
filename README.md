# JBoard

## 프로젝트 소개

본 프로젝트는 JSP를 활용하여 CRUD 기능을 제공하는 게시판 웹 애플리케이션입니다.

사용자는 게시글을 작성, 조회, 수정, 삭제할 수 있으며, 회원가입 및 로그인 기능을 통해 게시판 서비스를 이용할 수 있습니다.

이 프로젝트를 통해 JSP와 서블릿 기반 웹 애플리케이션 개발의 기본 개념을 익히고, 실제 게시판 기능을 구현하며 웹 개발의 흐름을 학습하는 것을 목표로 합니다.

## 주요 기능

- 회원가입
- 로그인
- 로그아웃
- 회원정보 조회 및 수정
- 게시글 목록 조회
- 게시글 작성
- 게시글 상세 조회
- 게시글 수정
- 게시글 삭제
- 게시글 검색

## 사용 기술

- Java
- JSP
- Servlet
- HTML
- CSS
- JavaScript
- MySQL
- Tomcat

## 프로젝트 목적

JSP와 서블릿을 기반으로 웹 애플리케이션의 기본 구조를 이해하고, MVC 패턴을 적용하여 게시판 기능을 구현하는 것을 목표로 합니다.

또한 회원 기능과 게시글 CRUD 기능을 직접 구현하면서 데이터베이스 연동, 요청 처리, 화면 출력 방식에 대한 실전 경험을 쌓을 수 있습니다.
